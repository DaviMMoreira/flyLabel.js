FloatLabel.js

This is the FloatLabel.js demo. The CSS defines the behavior of the labels. Behaviors authored by Mike Anderson unless otherwise stated.

classic

.flabel__label--classic {
  opacity: 0;
  top: 2em;
}
.flabel__label--classic.flabel__label--active {
  opacity: 1;
  top: 1em;
}

drop

.flabel__label--drop {
  opacity: 0;
  text-shadow: 0 0 25px orange;
  -webkit-transform: scale(11);
  -moz-transform:    scale(11);
  -o-transform:      scale(11);
  transform:         scale(11);
  z-index: -1;
}
.flabel__label--drop.flabel__label--active {
  opacity: 1;
  text-shadow: 0 0 0px orange;
  -webkit-transform: scale(1);
  -moz-transform:    scale(1);
  -o-transform:      scale(1);
  transform:         scale(1);
  z-index: 1;
}

from-right

.flabel__label--from-right {
  left: 5em;
  opacity: 0;
}
.flabel__label--from-right.flabel__label--active {
  left: 0%;
  opacity: 1;
}
